The **Smart Password Generator** is an interactive web-based tool that helps users create strong, secure, and personalized passwords.

## ğŸ’¡ Solution Approach
The Smart Password Generator solves this by:
- Taking user details (name, email, mobile number, etc.) as input.  
- Randomly selecting:
  - **2 characters** from the first name  
  - **2 characters** from the last name  
  - **6 random characters** from the remaining data pool  
- Generating a **10-character secure password**.  
- Displaying **real-time password strength** using a color-coded progress bar.  
- Providing **AI-like feedback messages** to guide the user.  
- Offering **interactive features** like show/hide password, copy to clipboard, and reset options.
